# lala-map

A visualisation challenge to interact with lalamove's mockAPI.

### how to use

    git clone git@github.com:denistsoi/here-be-dragons.git
    cd here-be-dragons
    npm install
    npm start


### Peer Dependancies

    git clone git@github.com:denistsoi/odyssey.git
    cd odyssey
    npm install
    npm start
